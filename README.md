# Soda Vanguardia — Landing Page

Landing page oficial de **Soda Vanguardia**, sodas artesanales elaboradas con agua de las Sierras de Córdoba.

## Stack
HTML5 + CSS3 + Vanilla JS — sin frameworks, sin dependencias, sin backend.

## Estructura
```
sodavanguardia/
├── index.html
├── css/styles.css
├── js/main.js
└── imgs/           ← todas las imágenes del producto
```

## Deploy en GitHub Pages
1. Subir el repo a GitHub (público)
2. Settings → Pages → Branch: `main` → Folder: `/ (root)` → Save
3. URL: `https://TU_USUARIO.github.io/sodavanguardia/`

## Deploy en Vercel (alternativa)
1. vercel.com → New Project → importar desde GitHub
2. Framework Preset: **Other**
3. Deploy → URL automática tipo `sodavanguardia.vercel.app`
